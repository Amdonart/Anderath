Hosting über GitHub Pages

Diese Website wird über GitHub Pages bereitgestellt, einen Dienst der GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA. Beim Aufruf der Website verarbeitet GitHub technisch erforderliche Nutzungs- und Zugriffsdaten. Dazu können insbesondere IP-Adresse, Geräte- und Browserinformationen, Datum und Uhrzeit der Anfrage, aufgerufene Seiten sowie die zuvor besuchte Website gehören. Die Verarbeitung dient der Bereitstellung, Stabilität und Sicherheit des Dienstes.

GitHub verarbeitet Daten unter Umständen in den USA und weiteren Ländern. Nach eigenen Angaben verwendet GitHub für internationale Datenübermittlungen unter anderem die Standardvertragsklauseln der Europäischen Kommission und nimmt am EU-US Data Privacy Framework teil. Weitere Informationen findest du in der Datenschutzerklärung von GitHub ↗.

Google Fonts

Für die Darstellung der Schriftarten stellt dein Browser eine Verbindung zu Google Fonts her. Dabei können insbesondere deine IP-Adresse und technische Browserdaten an Google übertragen werden.

Externe Links und Formulare

Die Website verlinkt unter anderem auf Discord, Twitch, Google Forms, CurseForge und den externen Impressumsdienst. Erst wenn du einen solchen Link öffnest, gelten die Datenschutzbestimmungen des jeweiligen Anbieters. Externe Seiten werden in einem neuen Tab geöffnet.

Modpack-Download

Das Anderath-Modpack wird als ZIP-Datei direkt von dieser Website angeboten und ist für alle verfügbar. Beim Download können dieselben technisch erforderlichen Serverdaten wie beim normalen Seitenaufruf anfallen.

Cookies und Tracking

Die Anderath-Website setzt selbst keine Analyse- oder Trackingdienste ein und speichert keine eigenen Cookies. Verlinkte externe Dienste können eigene Technologien verwenden, sobald du deren Seiten aufrufst.
